# Simple Blog Pro - Custom WordPress Theme

A modern, high-performance, and fully responsive WordPress theme built with **Bootstrap 5** and **Swiper JS**. This theme is specifically engineered for professional blogs and service-oriented websites.

## 🚀 Features
- **Dynamic Featured Slider:** Smooth, touch-enabled hero slider powered by Swiper JS.
- **Bootstrap 5 Integration:** Utilizes the latest Bootstrap framework for a mobile-first, responsive layout.
- **Custom Widget Areas:** Pre-configured support for "About the Author," Contact Info, and multi-column footers.
- **Optimized Navigation:** Clean, Bootstrap-based sticky navbar with support for dynamic WordPress menus.
- **Developer-Friendly Code:** Lightweight, well-documented PHP structure following WordPress coding standards.

## 🛠 Tech Stack
- **CMS:** WordPress (Core)
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **CSS Framework:** Bootstrap 5.x
- **Libraries:** Swiper JS (Slider), Bootstrap Icons
- **Backend:** PHP
